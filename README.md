介绍 ：第一个完整的GoWeb项目
1. 来源于哔哩哔哩尚硅谷goweb教程的图书商城项目，视频链接：https://www.bilibili.com/video/BV1nJ411D7P4
2. 源代码中的图书图片只有默认图片，已添加与图书名称对应的图片。
3. 原有sql文件缺少很多表，并且sql表中与图书对应的图片为默认图片，已修改sql文件为与图片一一对应

5. 可在原有功能上添加新功能。
